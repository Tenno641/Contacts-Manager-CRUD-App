# Contacts-Manager-CRUD-App
A simple Contacts Manager application that allows users to Create, Read, Update, and Delete (CRUD) contacts. Built to practice backend fundamentals, clean architecture, and database operations.

Features

- Add new contacts

- View all contacts

- View contact details

- Update existing contacts

- Delete contacts

- Server-side validation

---

RESTful API design

Tech Stack

- Backend: ASP.NET Core

- Language: C#

- Database: SQL Server 

- ORM: Entity Framework Core

- Architecture: Clean Architecture (Domain, Core, Infrastructure, Web)

- Tools: .NET 10
